批色助手 — 拖到收藏栏即装

把下面这个绿色按钮 拖 到浏览器的收藏栏（书签栏）：

📋 批色助手

使用
  1. 在 sso.geiwohuo.com 或 vocpub.dotfashion.cn 任意页面，点一下收藏栏的"批色助手"
  2. F12 打开 console，会看到「✅ 批色助手已就绪」
  3. 用法： 
    // 在 sso.geiwohuo.com（批色管理页）
pibao.list()                                        // 拉待批色清单

// 在 vocpub.dotfashion.cn（点"去自批"会打开此域）
pibao.detail('MLSH-29080605-001', 'M01326543')      // 看一条详情
pibao.dryRun('MLSH-29080605-001', 'M01326543')      // 不真发，只打 body
pibao.save('MLSH-29080605-001', 'M01326543')        // 真保存草稿（可被覆盖）
pibao.submitPass('MLSH-29080605-001', 'M01326543')  // 真审核通过（不可逆）

// 批量
pibao.batchSubmit([
  {orderNo:'MLSH-29080605-001', sku:'M01326543'},
  {orderNo:'MLSH-29080437-001', sku:'M01154021', weight: 220},
], { reallySubmit: true })
已 verified（2026-04-28）： list / detail / materialInfo / save 4 个 API 全部跑通。
对 MLSH-29080605-001 / M01326543 跑过一次 save，服务端返回 {code:0, msg:"success"}，记录从 version 1 → 2，bomPmState 1 → 2，weightMeasureValue 0 → 220.00，且仍在待批列表（save 不审核，符合预期）。
风险等级：
🟢 list / detail / dryRun / materialInfo — 完全只读 / 不真发
🟡 save — 写入草稿但不进入审核流程，可被覆盖
🔴 submitPass / batchSubmit(reallySubmit:true) — 真审核通过，记录从待批列表消失，对外可见
首次真跑前先 dryRun 看 body，确认无误再 submitPass 一条试试。
